• <p> tag is used to separate paragraphs in site
  • <br> tag is used as a line break in the html document without manually insert new line with enter key
  • <div> tag is used to differentiate different part of a site. The styling attributes for this element are font-size, float, margin, padding, ...
  • <ul> tag, or unordered list, is used to wrap the <li> tag, creating a list. This list does not provide a numbered list but bullet points, <ol> may be used to achive this. text-decoration style may be used to change how the list look
  • <table> tag to create variety of tables like timetable, exam, and curriculum. The element may use border, font-size, and font-weight attributes.
  • <tr> tag to create a single row for the table. Similar attributes in table can also be used in <tr>
  • <td> tag represent the column within a row in the table. This element can use width, height cospan, rowspan, etc. for attributes
  • <strong> tab is used to highlight importance of a text, or can be used simply to bold text, similar to <b> tag
  • <h1> tag, usually to idicate heading for a site, or just simply a large bolded text which is used to grab user's attention
  • <sup> tag can be used to superscript a text, which can be used to indicate trademark or reference link
